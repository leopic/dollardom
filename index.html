$dom

This page needs Javascript to work.

5k Javascript library for selecting, styling, traversing and animating DOM elements.

What can you do with $dom:

  • Create elements.
  • Animate elements.
  • Add/Remove/ToggleClasses.
  • Chain methods, ala jQuery.

Go through our API and see all that you can do.

Why $dom?

  • It's small: 2.3 kB for the minified/gzipped version.
  • It's general: it's not mobile-only or desktop-only. It's perfect for mobile-first or responsive design.
  • It doesn't include the features you don't need, e.g. Array iteration functions (forEach, map, reduce, etc.) or Ajax.
