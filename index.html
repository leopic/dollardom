$dom

5k Javascript library for selecting, styling, traversing and animating DOM elements.

What can you do with $dom:

  • Create elements.
  • Animate elements.
  • Add/Remove/ToggleClasses.
  • Chain methods, ala jQuery.

All in less than 5k in a fully-tested cross-browser IE6+ library. checkout more examples.

Why $dom?

  • Create elements.
  • Animate elements.
  • Add/Remove/ToggleClasses.
